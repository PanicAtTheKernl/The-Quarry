Allowed Duplication Methods

The Quarry embraces certain Minecraft mechanics as legitimate advanced techniques for technical gameplay.

Technical Philosophy: Unintended mechanics are considered advanced features that enhance technical gameplay experience.

Gravity Block Duplication

  • Sand (all variants including suspicious sand)
  • Gravel (including flint drops)
  • Concrete Powder (all 16 colors)

Rail Duplication

  • Regular Rails (standard minecart tracks only)

Carpet Duplication

  • All Carpet Colors (all 16 variants)

TNT Duplication

  • TNT Blocks (for world eaters, perimeters, technical contraptions)

Explicitly Prohibited

  • Dragon Egg (unique item must remain singular)
  • Powered Rails (balance reasons)
  • Detector Rails (balance reasons)
  • Activator Rails (balance reasons)
Fair Use Policy: Use responsibly to avoid server lag or crashes. Consider impact on other players.
